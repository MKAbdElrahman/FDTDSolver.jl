FDTDSolver.optimal_cell_size — Method

function optimalcellsize(Δmin, Δmax, xc; nparticles = 100, p_norm = Inf)

Uses particle swarm optimization to find a cell size in the range [Δmin,Δmax] such that the nodes x_c are integer multiples of this cell size.

...

Arguments

  • Δ_min: minimum cell size
  • Δ_max: maximum cell size
  • x_c: constraint node coordinates
  • n_particles: number of particles in the swarm (default: 100)
  • p_norm: norm used to calculate the misfit objective function (default: Inf)

...
